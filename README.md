# Transaction Management System


CLI localhost command

- git clone https://github.com/kartikey-shivam/Transaction-Dashboard-backend
Add .env file
- MONGO_URI=your-mongobd-url
- JWT_SECRET=your-secret-key
- PORT=5000

CLI command
- npm install
- npm start
